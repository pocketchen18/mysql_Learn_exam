# MySQL 考试练习系统 - 使用说明

本系统是一个基于 FastAPI 和 React 开发的 MySQL 题目练习工具，支持随机刷题、错题重做、统计分析等功能。

## 1. 快速启动
1. 找到并双击运行 `MySQL考试练习系统.exe`。
2. 程序启动后会通过后台开启 Web 服务，并自动在你的默认浏览器中打开练习界面（通常是 http://127.0.0.1:8000）。

## 2. 核心功能
- **随机练习**：支持按题目类型（单选、多选、填空、判断）和分类进行筛选练习。
- **错题本**：系统会自动记录你做错的题目，方便后续针对性复习。
- **题目管理**：支持手动添加、修改、删除题目。
- **批量导入**：支持通过 JSON 格式批量导入题目数据。

## 3. 注意事项
- **数据存放**：请确保 `question_bank` 文件夹与 `.exe` 文件处于同一目录下，否则无法加载题库。
- **进度保存**：你的练习进度和错题记录保存在 `user_progress.json` 文件中，请勿删除该文件，否则进度会丢失。
- **浏览器兼容性**：建议使用 Chrome、Edge 或其他现代浏览器以获得最佳体验。

## 4. 常见问题
- **运行报错**：如果运行报错，请检查是否已有相同端口（8000）的服务在运行。
- **浏览器未打开**：如果运行后浏览器没有自动打开，请手动在浏览器地址栏输入 `http://127.0.0.1:8000` 访问。
