# 项目详情

**MySQL 考试练习系统** 是一款专为准备数据库相关考试的用户打造的桌面化 Web 应用。它集成了题目管理、随机练习、错题记录以及数据统计等多项功能，通过现代化的 UI 设计提供流畅的刷题体验。

该项目采用前后端分离架构，后端使用高性能的 **FastAPI** 处理数据逻辑，前端使用 **React** 和 **Tailwind CSS** 构建响应式界面，并支持通过 **PyInstaller** 一键打包为 Windows 可执行文件，实现零环境依赖运行。

## 核心功能

- **🎯 智能刷题**：支持按单选、多选、判断、填空等题型筛选，支持随机乱序练习。
- **📝 错题强化**：自动收集用户答错的题目，建立个性化错题本，支持专项突破。
- **📊 进度看板**：可视化展示答题正确率、各分类掌握程度，学习进度一目了然。
- **🛠️ 题库管理**：内置可视化界面，支持随时对题目进行增删改查，并支持 JSON 批量导入。
- **📦 离线运行**：支持打包为本地 EXE，无需安装 Python 环境即可在 Windows 上直接使用。

## 技术亮点

- **前后端解耦**：标准 RESTful API 设计，前端 Vite 构建。
- **数据持久化**：采用轻量级 JSON 存储方案，无需配置复杂的数据库服务器。
- **心跳监控**：后端集成自动监控机制，前端关闭后服务自动释放资源。
